### Start
While in the directory with the .sh file,
type the command without quotes "chmod +x run.sh"
and the command to run "./run.sh"

### Creating a Shortcut 
**Linux / Ubuntu:**  
1. Go to the folder with your `.sh` script.  
2. Right-click → Create Shortcut / Make Link → drag to Desktop.  
3. Right-click → Properties → Permissions → Allow executing as program.  
4. Double-click to run in Terminal.  

**macOS:**  
1. Open Automator → New Application → Run Shell Script.  
2. Paste `/path/to/run_dictionary.sh`.  
3. Save → drag to Desktop.  
4. Double-click to run.
